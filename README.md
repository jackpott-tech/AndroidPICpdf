# AndroidPICpdf

Offline Android app for creating clean A4 PDFs from photos.

## Highlights
- 100% local workflow (no cloud, no upload, no tracking)
- Multi-photo import
- Reorder/delete photos before export
- 4/5/6 photos per page layouts
- PDF export for sharing/printing

## Repository content
This repo currently provides debug APK builds.

Latest build:
- `AndroidPICpdf-1.0.apk`

## Privacy
The app processes data locally on-device.
