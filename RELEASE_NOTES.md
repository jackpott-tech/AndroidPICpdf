# Release Notes

## v1.0 (2026-02-22)

Erste stabile Version von **AndroidPICpdf**.

### Highlights
- Lokale/offline Foto-zu-PDF-App (keine Cloud, kein Upload)
- Projektverwaltung (anlegen, öffnen, löschen)
- Bilder importieren und in Seiten strukturieren (4/5/6 pro Seite)
- Bildreihenfolge bearbeiten und Bilder löschen
- Seitenüberschriften + Bildunterschriften
- PDF-Export (A4, druckfreundlich)
- Aufgeräumtes UI mit einklappbaren Einstellungen im Editor
- Datenschutz-Hinweise als eigene Seite in der App

### Datenschutz
- Alle Daten bleiben lokal auf dem Gerät
- Keine Telemetrie / kein Tracking
